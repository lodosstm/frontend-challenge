
установка:

  npm install

запуск:

  npm run
  
  открывается по адресу  http://localhost:8888/  
  
тесты:

  npm run test



Для работы приложения используется JSON Server.
файл с данными для базы расположен ./db/db.json ,
можно создать новый со следующим содержанием: {"users": []}
